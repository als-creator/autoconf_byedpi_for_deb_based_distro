# Установка и запуск ByeDPI на Alt Linux

Этот скрипт автоматизирует установку, настройку и запуск сервиса ByeDPI на Alt Linux и других deb based дистрах, учитывая отсутствие `sudo` и использование `su` для повышения прав.

---

## Особенности

- Запрет прямого запуска от root: рекомендуется запускать скрипт через `su -c`.
- Проверка наличия команды `su`.
- Автоматический поиск конфигурационного файла `byedpi.conf` в `/etc/` или `/usr/local/etc/`.
- Установка пакета `byedpi` из репозитория с помощью пакетного менеджера (`apt-get` или `rpm`) через `su`.
- Настройка параметров ByeDPI в конфигурационном файле.
- Автоматическое включение и запуск сервиса через `systemctl` с использованием `su`.

---

## Быстрый старт

1. Сохраните скрипт в файл, например `autoconf_byedpi_for_deb_based_distro.sh`.
2. Сделайте скрипт исполняемым:

chmod +x autoconf_byedpi_for_deb_based_distro.sh

text

3. Запустите скрипт с помощью `su`:

su -c ./autoconf_byedpi_for_deb_based_distro.sh

text

---

## Управление сервисом ByeDPI

- Перезапуск:

su -c 'systemctl restart byedpi'

text

- Запуск:

su -c 'systemctl start byedpi'

text

- Проверка статуса:

su -c 'systemctl status byedpi'

text

---

## Конфигурация

По умолчанию конфигурационный файл размещается по пути:

/etc/byedpi.conf

text

Если файла нет, скрипт ищет конфигурацию в `/usr/local/etc/byedpi.conf`. В файле можно изменить правила фильтрации, порт и другие параметры.

---

## Зависимости

- Наличие пакетного менеджера `apt-get` или `rpm`.
- Возможность использовать команду `su` для установки пакетов и управления сервисами.

---

## Важные замечания

- Не запускайте скрипт напрямую от root.
- Убедитесь, что у вас есть права на использование `su` и установку пакетов.
- Используйте скрипт на свой страх и риск.
